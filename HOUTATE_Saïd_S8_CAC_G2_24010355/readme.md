## 📊 Description du projet
**Titre** : *Formation en Machine Learning et Deep Learning - Atelier 1 : Comprendre la Classification Supervisée*

Ce projet Jupyter Notebook présente une **introduction pratique à la classification supervisée** couvrant :

### 🎯 **Objectifs pédagogiques**
- Exploration et visualisation des données
- Entraînement de modèles de classification supervisée
- Évaluation des performances (Accuracy, Precision, Recall, F1-score)
- Comparaison multi-modèles

### 📋 **Cas d'étude implémentés**
1. **Exemple simple** : Prédiction d'admission universitaire  
   *(Score Examen + Score Entretien → Accepté/Refusé)*
2. **Dataset réel** : **Breast Cancer Wisconsin**  
   *(569 échantillons, 30 features morphologiques → Bénin/Malin)*
